::::::: СИСТЕМА ОБРАБОТКИ СТАТИСТИКИ V0.2b :::::::

УСТАНОВКА :: Распаковать в корень любого диска: на HDD или флешку. На HDD работает намного быстрее. Можно распаковать в любую папку на HDD и создать из этой папки диск subst'ом.

СТАРТ :: В начале запускаешь PStart. Он стартует всю систему. В конце запускается Internet Explorer c адресом http://www.sos.ka2 . Если после запуска браузера он с сайтом не соединился, то набираешь адрес в адресной строке вручную. После старта можно любым браузером заходить естественно. Можно сделать для флешки autorun.inf, но туда любить лезть всякая гадость и я его удалил (не должно его быть).

АВТОРИЗАЦИЯ :: логин - niculin, пароль - niculin

ПРАВКА СПИСКА СЧЕТОВ :: кликаешь на батнике в корне диска и правишь. Там наши старые счета записаны. Вводишь свои по образцу.

ВРАЧИ :: В правом верхнем углу сайта есть ссылочка на phpMyAdmin. Открываешь базу sos2009, таблицу mperson и правишь ее ручками. То есть добавляешь записи своих врачей. Первое поле служебное (автоинкремент) не заполнять, второе - код врача, третье - ФИО, четвертое - номер специальности. Ненужных врачей в принципе можно удалить (замочить :D ), чуть быстрее работать будет.

ЗАВЕРШЕНИЕ РАБОТЫ :: !!!!!! Правой кнопкой в трее кликнуть значок PStart и выбрать пункт Exit !!!!!! Только так, а то имеешь возможность повредить базу !!!!!!
